#  Refactoring Project

##  Опис проєкту

Даний проєкт присвячений рефакторингу існуючого серверного застосунку для онлайн-опитувань.  
Метою є покращення якості коду шляхом застосування сучасних принципів розробки, підвищення читабельності, підтримуваності та масштабованості.

---

##  Мета роботи

- проаналізувати існуючий код
- знайти проблеми (code smells)
- застосувати техніки рефакторингу
- покращити структуру проєкту
- додати юніт-тести

---

##  Технології

- Node.js
- Express.js
- MySQL
- Jest (для тестування)

---

##  Структура проєкту

```text
REF_PROJ_1
│
├── original/       # оригінальний код (до рефакторингу)
├── refactored/     # покращена версія
├── tests/          # юніт-тести
├── documents/      # звіт по рефакторингу
├── README.md
├── package.json
```

---

##  Виявлені проблеми

У початковому коді були знайдені наступні проблеми:

- великий монолітний файл
- змішування відповідальностей
- дублювання коду
- відсутність валідації
- слабка обробка помилок
- жорстко прописані налаштування
- SQL-запити в маршрутах

---

##  Виконаний рефакторинг

Було застосовано наступні техніки:

- Extract Module
- Separation of Concerns
- Extract Configuration
- Replace Duplicated Code
- Extract Function
- Move Function
- Introduce Validation
- Guard Clauses
- Centralized Error Handling
- Layered Architecture

##  Запуск проєкту

### 1. Встановлення залежностей

npm install


### 2. Налаштування середовища

Створити файл `.env` у папці `refactored`:

```
PORT=3000
DB_HOST=127.0.0.1
DB_USER=root
DB_PASSWORD=password
DB_NAME=OnlineSurveys
```
